Ctrl+x : exit nano.
Ctrl+w : where is a specific word.
Ctrl+k : cut from the current cursor position to the end of the current line.
Ctrl+a : move to the beginning of the current line.
Ctrl+e : move to the end of the current line.
Esc g : go to line number and column number.
Esc $ : soft wrapping of overlong lines.
Esc | : Moves cursor to first letter on first line.
-$ : softwrap (put entire content on one line)
-i : autoindent
